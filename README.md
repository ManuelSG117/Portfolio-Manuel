## Proyectos Destacados

### 📱 CAPASU
Aplicación móvil para el pago de servicios de agua del estado de Michoacán con **+1k descargas** en Google Play Store. Integración de pasarela de pagos, notificaciones en tiempo real, API RESTful y crashlytics.
- **Tecnologías:** Kotlin, Firebase, PHP, MySQL

### 📋 CotizApps
App de cotizaciones con catálogo de productos/clientes, generación de PDFs con logo y firma, reportes de evidencia con fotos, sincronización con Supabase y autenticación por email.
- **Tecnologías:** Kotlin, Supabase, Jetpack Compose

### 🚗 Flota CAPASU
Sistema web para gestión de flota vehicular con geocercas, notificaciones y reportes que redujeron costos operativos en un 10%.
- **Tecnologías:** PHP, Google Maps, Firebase, MySQL, Bootstrap

### 📊 Sistema de Reportes CAPASU
Plataforma web unificada para reportes ciudadanos con reducción del 40% en tiempo de respuesta y mapeo de atlas de riesgo.
- **Tecnologías:** PHP, JavaScript, MySQL, Bootstrap
